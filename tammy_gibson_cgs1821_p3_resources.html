This is a mix of websites I use to study specific languages. A quick search of google for html or css questions will usually bring up one of the below websites. On these site I can view code samples and access resources to troubleshoot many development issues.

  • Eloquent Javascript - A modern introduction to programming. This is a recommended book for those interested in learning Javascript.
  • Cplusplus.com - One of my favortie reference websites for questions regarding the programming language C++. Great forum support and tutorials.
  • CSS Tricks - Probably my second most visited site. Devoted to CSS with tutorials on various ways to maximize use of CSS. CSS Tricks also has a video and podcast with tutorials. Superb site by Chris Coyier.
  • Google Fonts - A great resource for fonts allowing you to see them grouped together. Includes directions for use on your website and samples of available text effects as well.
  • Michael Crump - A tutorial for setting up Git with Visual Studio to push your work to github. This was pretty important for me to learn so I could display my code work online.
  • Codepen.io - My favorite website, aside from google for finding code snippets. Registering on this site will allow you to host HTML, CSS and Javascript for the public to view. You can fork other projects into your account and work with the code as well.
  • Smashing Magazine - Online magazine for web developers. Up to date articles on trends in website technologies. Lots of topics covered here as well as information on various conventions.

If all of the above reference material was not enough to get you started then definately check out youtube! I prefer to learn by watching others if I can. Youtube is a great way to watch other people use developer environments and interact with code. I taught myself to use BASH command lines for git that way.
